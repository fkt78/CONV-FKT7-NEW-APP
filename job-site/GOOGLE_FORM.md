# Googleフォームの作り方（伊賀3店・求人・問い合わせ用）

Cursor や AI から **あなたの Google アカウントでフォームを自動作成することはできません**。次の手順でブラウザから作成し、できた **公開 URL** を `contact-form-url.js` の `GOOGLE_FORM_URL` に貼り付けてください。

## 1. 新規フォームをつくる

1. [Googleフォーム](https://docs.google.com/forms/u/0/) を開く（Googleにログイン）
2. **空白** から開始
3. フォームのタイトル例:
   - **セブン‐イレブン 伊賀エリア3店（平野北谷・平野東町・忍者市駅南）| 応募・お問い合わせ**

## 2. 説明文（フォーム冒頭）

以下を「説明」に貼り付けて調整してください。

```text
セブン‐イレブン 伊賀平野北谷店・伊賀平野東町店・伊賀忍者市駅南店の求人・パート応募、その他お問い合わせフォームです。
いただいた内容は担当より改めてご連絡します。お急ぎの際は店頭スタッフへもお声がけください。
```

## 3. 質問の例（「＋」で追加）

| 質問文言 | 形式の目安 | 備考 |
|----------|------------|------|
| お名前（フルネーム） | 記述式（短文） | 必須推奨 |
| ふりがな | 記述式 | |
| 年齢 | 記述式 / プルダウン | 未成年の場合は保護者同意が別途必要な場合あり |
| 電話番号 | 記述式 | 必須推奨 |
| メールアドレス | フォーム標準の「メールアドレス収集」または記述式 | 返信用 |
| ご希望の店舗 | ラジオボタンまたはプルダウン | 選択肢: 平野北谷店 / 平野東町店 / 忍者市駅南店 / どこでも可 |
| ご応募内容 | ラジオ | 例: アルバイト・パート応募 / 採用に関する質問のみ / その他 |
| 週○日・希望時間帯（例：平日夕方〜、土日中心 など） | 段落（長文） | |
| 接客・レジ・品出しなどの経験 | 段落 または ラジオ | |
| 質問・伝えておきたいこと | 段落 | 任意 |
| 個人情報の取り扱いへの同意 | チェックボックス | 必須。社内文面に合わせて同意文を記載 |

質問の順番や必須設定は、店舗の運用に合わせて変更してください。

## 4. 設定（歯車アイコン）

- **回答にメールアドレスを含める**：返信や通知に使うならオンが便利です。
- **回答者に確認メールのコピーを送信**：必要に応じてオン。
- **回答を編集**：運用に応じて。

通知をメールで受け取りたいときは、フォームの **「回答」タブ** でスプレッドシート連携やメール通知（拡張機能・Google Apps Script 等）を検討してください。

## 5. 公開して URL を取得

1. 右上の **送信** をクリック
2. **リンク** タブで **URL を短縮** をオンにすると `forms.gle/...` が使えます
3. 表示された URL をコピー

## 6. このリポジトリに反映

`job-site/contact-form-url.js` を開き、次のように **コピーした URL を文字列で代入** します。

```js
const GOOGLE_FORM_URL = 'https://docs.google.com/forms/d/e/xxxx/viewform'
// または
const GOOGLE_FORM_URL = 'https://forms.gle/xxxx'
```

保存後、`npm run deploy:job-site` で求人サイトを再デプロイしてください。

## 7. プレビューの見方（PC・スマホの長さの確認）

編集画面のまま、**実際の応募者と同じ見え方**を確認できます。

### パターンA：編集画面から（いちばん簡単）

1. フォームを開いた状態で、右上の **目のアイコン「プレビュー」**（「プレビュー」とツールチップが出るボタン）をクリック。
2. 新しいタブが開き、**送信後まで含まない**入力画面だけが表示されます。
3. **閉じる** と編集画面に戻ります。

### パターンB：スマホでそのまま見たい

1. 右上 **送信** → **リンク** タブで表示された URL を **スマートフォンにメールやLINEで送る** か、QRコード化して読み取る。
2. スマホのブラウザで開くと、**スマホ向けの縦長レイアウト**で確認できます。

※ まだ「送信」で一般公開していない場合でも、このプレビュー用リンクは多くの場合あなた自身なら開けます。公開設定は **設定（歯車）** で確認してください。

### パターンC：PCのブラウザで「スマホ幅」を再現

1. **プレビュー** を開いた状態で、キーボードの **F12**（Mac は `Fn + F12` など）で**開発者ツール**を開く。
2. **デバイスツールバー**（スマホ・タブレットのアイコン）をオンにする。
3. 画面上部で **iPhone や Pixel** など端末を選び、縦スクロールの長さや文字の改行を確認する。

時間帯の選択肢が多いフォームは、**スマホでどこまでスクロールが必要か**だけ一度見ておくと安心です。

## 8. 個人情報の取り扱い・保護者同意の足し方

**文言は法的助言ではありません。** 店舗の方針・顧問（社労士・行政など）に確認してから差し替えてください。

### 8-1. 「個人情報の取り扱い同意」（全員・必須）

1. **＋** で質問を追加する。
2. 形式を **チェックボックス** にする。
3. **質問タイトル** の例:

```text
個人情報の取り扱いについて
```

4. **説明**（質問タイトルの下の小さな欄）に、次のような文を貼り、店舗名・運営者名を実名に変えてください。

```text
ご記入いただいた個人情報は、採用選考・お問い合わせへの対応・連絡のために利用し、当該目的の範囲内でのみ管理します。第三者に提供する場合は法令に基づく場合を除き、あらかじめご本人の同意を得ます。内容の開示・訂正・利用停止等のお申し出については、店舗窓口にて対応します。
※ 詳細は店頭のプライバシーポリシー等をご確認ください（必要に応じて URL を記載）。
```

5. **選択肢**を 1 つだけにする:

```text
上記を確認し、個人情報の取り扱いに同意します
```

6. 右下 **必須** をオンにする。

### 8-2. 「保護者同意」（未成年が応募し得る場合）

次の**どちらか**（または両方）を採用できます。

#### 方式A：シンプル（1問追加・運用でカバー）

1. 形式 **チェックボックス**、**必須**。
2. 質問タイトル例:

```text
未成年者の応募について（保護者の同意）
```

3. 説明例:

```text
18歳未満の方は、必ず保護者（親権者等）の同意を得たうえでご応募ください。18歳以上の方は、このままチェックを入れてください。
```

4. 選択肢（1つ）:

```text
18歳以上である、または保護者の同意を得ている
```

※ 18歳以上も**同じチェック**で進めるので、実質「誓約」に近い扱いになります。より厳密に分けたい場合は方式Bへ。

#### 方式B：18歳未満だけ保護者氏名を聞く（セクション分岐）

1. **生年月日**や**年齢**の質問の**後**に、新しい **セクション** を追加（右サイドバーの「セクションを追加」）。
2. セクションタイトル例: `保護者様への確認（18歳未満の方）`
3. その**直前**に **ラジオボタン** の質問を置く:

   - 質問: `年齢区分`
   - 選択肢: `18歳以上` / `18歳未満`
   - **⋮（その他）** メニュー → **回答に応じてセクションに移動**（UI の名称は環境で若干異なります）で、  
     - 18歳以上 → **次のセクション**（保護者セクションを**スキップ**する先、例：希望シフトの質問があるセクション）  
     - 18歳未満 → **「保護者様への確認」セクション**

4. 保護者セクション内に **記述式** を並べる例:
   - `保護者氏名（フルネーム）` 必須
   - `保護者との続柄` 必須
   - `保護者連絡先（電話番号）` 必須

5. 保護者セクションの**最後**の質問からは、**次のセクション**へ合流するよう **セクションの行き先** を設定。

Googleフォームのメニュー表記は英語 UI だと "Go to section based on answer" です。日本語でも「セクションへ移動」系の項目を探してください。

### 8-3. 注意

- **個人情報保護方針・採用おける取り扱い**は会社・店舗の実態に合わせる必要があります。ここに書いた文は**叩き台**です。
- アルバイト採用で**労働条件通知**など法令上別途必要な書面がある場合は、フォームとは別の手順で対応してください。
